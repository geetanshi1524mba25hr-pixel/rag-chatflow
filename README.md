# RAG Chatflow Project

## 📌 Overview

This project demonstrates a Retrieval-Augmented Generation (RAG) chatbot built using Flowise. The chatbot retrieves relevant information from documents and generates accurate responses using embeddings and a conversational model.

## 🚀 Features

* Document-based Question Answering
* Conversational Memory
* Vector Store Retrieval
* Embedding-based Search
* Flowise Chatflow Implementation

## 🛠️ Tech Stack

* Flowise
* Mistral AI / LLM
* Google Gemini Embeddings
* In-Memory Vector Store
* Recursive Text Splitter

## 📂 Workflow

1. Load document (PDF)
2. Split text into chunks
3. Generate embeddings
4. Store in vector database
5. Retrieve relevant context
6. Generate answer using LLM

## 🧠 RAG Architecture

User Query → Retriever → Vector Store → Context → LLM → Response

## ▶️ How to Run

1. Open Flowise
2. Import the chatflow
3. Upload document
4. Ask questions in chat window

## 🎯 Use Cases

* Document Chatbot
* Knowledge Base Assistant
* PDF Question Answering
* AI Helpdesk
